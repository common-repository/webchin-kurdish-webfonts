####English

####Webchin fonts
Webchin package webfonts for wordpress, consists of 98 pieces of designer wordpress themes, and you can choose a special font for line by line in a for Posts. Or you can select specific fonts for dashboard for better looking.

this plugin is written in a way where it can be in the theme of the slightest create a file under the name <strong>helbest.css</strong> in the theme directory, where you can edit without having to change any original Them file. For when you wanted to change the theme to be taken lose nothing.

== Installation ==

This plugin follows the [standard WordPress installation method][]:

1. Upload the `webchin-kurdish-webfonts` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to your visual editor to begin using your webchin webfonts.

[standard WordPress installation method]: http://codex.wordpress.org/Managing_Plugins#Installing_Plugins
